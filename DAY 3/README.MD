# Day 3 - SQL JOINS

## Project Overview

This is Day 3 of my SQL learning journey. In this project, I practiced SQL JOINS by working with employees and departments tables.

## Work Done

* Created two related tables
* Inserted sample data
* Applied different types of JOINS
* Performed analysis using aggregate functions

## Concepts Covered

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* GROUP BY
* Aggregate functions (COUNT, SUM, AVG)

## Key Analysis

* Employee count per department
* Total and average salary
* Highest paid employee in each department

## Learning Outcome

This project helped me understand how to combine and analyze data from multiple tables using SQL.
